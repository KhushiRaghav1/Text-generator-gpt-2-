# 🤖 GenAI Text Generator using GPT-2

## 📌 Description

This project demonstrates a basic implementation of **Generative AI** using a pretrained **GPT-2 language model** from the Hugging Face Transformers library. The model generates human-like text based on a user-provided input prompt by predicting the next sequence of words using probabilistic language modeling.

The system takes a starting sentence as input and produces a coherent continuation, showcasing how modern Large Language Models (LLMs) can be used for text generation tasks such as content creation, storytelling, and automated responses.

---

## 🚀 Features

* Generates meaningful text from user prompts
* Uses pretrained GPT-2 (no training required)
* Demonstrates core concept of next-word prediction
* Simple and beginner-friendly implementation
